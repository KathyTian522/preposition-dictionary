for (preposition)

1. to express Recipient function

1a

When you describe a state (such as an emotion, concern, or evaluation) and specify whom or what this state affects, for is used to designate the people or situation affected.

USAGE NOTE: State [n|adj.] + for + Participant | State [n]

Only the most expensive is good enough for some new parents.

The Committee is responsible for planning.

1b

When an action is performed and a particular object or person is concerned or benefited from it, for is used to represent the object or person.

USAGE NOTE: Action [v] + ([n]) + for + Participant | Beneficiary [n]

The students are waiting for you in the classroom.

The document lists three options for local authorities.

1c

This structure highlights a message or decision and its relevance to a specific entity.

USAGE NOTE: (Participant [n]) + for + Participant [n]

This is good news for taxes.

2. to express Purpose function

When you do something, for is used to show the purpose or the intended outcome.

USAGE NOTE: Action [v|n|-ing] + for + Goal | State | Participant [n|-ing]

They use speed cameras for the purpose of improving road safety.

He does a hundred steps up and down for his fitness.

3. to express Extent function

Used to express the time, degree, or range in which a state continues or an action performs.

USAGE NOTE: State | Action | Measure [v|n|adj.] + for + Extent [n]

It feels like we’ve been running for miles.

She appeared to absorb for a few moments.

4. to express Cause function

When something happens because of a reason, for introduces that cause.

USAGE NOTE: State [adj|v] | Action [v] + for + Reason [n|-ing]

He was sorry for the result.

She received the award for her part in discovering a cure.

5. to express Exchange function

For indicates something given, sold, or paid in return for something else.

USAGE NOTE: Action [v] + Participant [n] | Measure [n] + for + Measure | Participant [n]

We had to pay three pounds for the minibus journey.

Firefighters will swap hoses for walking boots.
